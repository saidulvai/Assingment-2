var:

Scope: Variables declared with var are globally scoped or function/locally scoped.

Hoisting: var variables are hoisted to the top of their scope and initialized with a value of undefined.

let:

Scope: Variables declared with let have block-level scope. They are limited to the block defined by curly braces {}.

Hoisting: Like var, let variables are hoisted, but they are not initialized until their declaration is encountered.

const:

Scope: Variables declared with const also have block-level scope.

Immutable: const variables cannot be re-assigned or re-defined after their initial assignment.

undefined:

Definition: When a variable is declared but not initialized, or when a function does not return a value, the variable or the function’s result is undefined

null:

Definition: null represents the deliberate absence of any object value. It’s used to explicitly indicate that a variable or object property should have no value or no reference to any object.
